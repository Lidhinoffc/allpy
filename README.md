# Allkitpy

Allkitpy is an all-in-one Python utility library.

## Features

- 🖼 Image Processing
- 📄 PDF Tools
- 🎵 Audio Utilities
- 🎬 Video Utilities
- 🌐 Web Utilities
- 📊 Data Processing
- 💻 System Utilities
- 🤖 AI Providers

## Installation

```bash
pip install allkitpy
```

## Example

```python
import allkitpy

print(allkitpy.text.uppercase("hello"))
```

Output:

```
HELLO
```

Python 3.10+
