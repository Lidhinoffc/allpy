# Contributing to Allpy

Thank you for your interest in contributing.

## Requirements

- Python 3.10+
- Virtual Environment

## Setup

```bash
pip install -e .
```

## Run Tests

```bash
pytest
```

## Coding Style

- Follow PEP 8.
- Use descriptive names.
- Add type hints when possible.
- Write tests for new features.

## Pull Requests

Please ensure all tests pass before submitting changes.
